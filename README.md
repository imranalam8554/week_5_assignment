# Week 5 - Apache Spark Fundamentals and Data Processing

## Objective

The main objective of this assignment is to learn the basics of Apache Spark and understand how it can be used to clean, transform, filter, and analyze large datasets using PySpark DataFrames.

---

## Tools Used

* Python
* Apache Spark (PySpark)
* Jupyter Notebook

---

## Dataset

* Sample Superstore Dataset (CSV)

---

## Work Completed

In this assignment, I:

* Created a Spark Session using PySpark.
* Loaded the Superstore CSV file into a Spark DataFrame.
* Explored the dataset by checking its schema.
* Removed duplicate records from the dataset.
* Handled missing values using Spark functions.
* Applied filters to retrieve specific records.
* Performed aggregation operations such as average, count, minimum, and maximum.
* Grouped data using the `groupBy()` function.
* Learned how Spark DataFrames work and how immutability affects data transformations.
* Understood the concept of shuffle operations during grouping.

---

## Concepts Learned

* Apache Spark Fundamentals
* In-Memory Processing
* Spark DataFrames
* Data Cleaning
* Data Transformation
* Filtering Data
* Aggregation Functions
* GroupBy Operations
* Shuffle Operations
* Schema Handling

---

## Results

The assignment demonstrates how Spark can efficiently process data using DataFrame operations. Different cleaning, filtering, grouping, and aggregation techniques were applied to understand how Spark handles large datasets.

---

## Conclusion

Through this assignment, I gained practical experience with Apache Spark and PySpark DataFrames. I learned how to clean and transform data, perform aggregation, and use Spark's distributed processing capabilities to work with datasets more efficiently. This assignment also helped me understand important Spark concepts such as immutability, in-memory processing, and shuffle operations.
